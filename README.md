# KryptoPay SDK

TypeScript SDK for integrating with KryptoPay services.

## Quick start

1. Install dependencies

```bash
npm install
```

2. Build

```bash
npm run build
```

3. Use

Import and use the SDK from `src/` or the built package as appropriate for your project.
